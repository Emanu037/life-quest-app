Dear Diary - Readme

Este projeto consiste num sistema de bem-estar e produtividade desenvolvido em Flutter. Ele foi criado para transformar a rotina em um momento de leveza e acolhimento.

O sistema une três diferentes funcionalidades:

    Mascote 2D: Pet personalizável. Use a loja para comprar acessórios exclusivos com suas economias de peixes.

    Diário: Um espaço seguro para registrar suas memórias, humores e reflexões diárias.
    
    Lista de Tarefas Gamificada: Transforme suas obrigações em missões. Ao concluir suas tarefas do mundo real, você ganha peixes para cuidar do seu amigo virtual.



Tecnologias Utilizadas

    Linguagem: Dart (v3.0.0+)

    Framework: Flutter (v3.10.0+)

    Gerenciamento de Estado: Provider (biblioteca para conectar a lógica do AppViewModel à interface)

   
Pré-requisitos do Sistema
Antes de tentar abrir o projeto, seu computador precisa ter:

    Flutter SDK: Instalado e configurado no PATH do sistema.

    Android Studio.

  

Configuração e Instalação
    1. Extraia a pasta .zip do projeto

    2. Instalar Dependências
    Dentro da pasta raiz do projeto, execute o comando para baixar a biblioteca Provider e outras dependências listadas no pubspec.yaml:

    Bash
    flutter pub get


    Como Rodar no Emulador (Android Studio)
    Abrir o Android Studio: Vá em File > Open e selecione a pasta raiz do projeto Flutter.

    Criar Emulador (AVD):

    Clique no ícone Device Manager (lado direito superior).

    Clique em Create Device.

    Escolha um dispositivo (ex: Pixel 7).


    Rodar o App:

    Com o emulador ligado, no terminal do VS Code ou do Android Studio, digite:

    Bash
    flutter run
    Ou pressione F5 se estiver usando o VS Code com a extensão do Flutter instalada.
